Hardware:
- MOS Technology 6502 processor
- 16bit address bus (addressing at most 64Kb of memory)
- Little endian
- Zero page addressing
- 256B stack memory
Registers:
- Program counter (PC), 16bit
- Stack pointer   (SP), 8bit
- Accumulator     (A),  8bit
- Index register X,     8bit
- Index register Y,     8bit
- Processor status (PS),8bit
